# Data

## raw

Unveraenderte Rohdaten. Dateien in diesem Ordner sollten nicht manuell ueberschrieben werden.

## processed

Bereinigte oder transformierte Daten. Diese Dateien koennen aus Skripten oder Notebooks erzeugt werden.
